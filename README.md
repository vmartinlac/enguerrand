ENGUERRAND
==========

Dependencies
------------

* Ceres
* Eigen3
* OpenCV
* openscenegraph
* Qt5
* realsense2
* Sophus
* Threading Building Blocks

Added value
-----------

* Implementation of various SLAM methods:
    * EKF SLAM
    * Graph SLAM
    * Fast SLAM
* Custom algorithm to track ping-pong balls in video stream.
* User interface to inspect pose and map provided by SLAM algorithms.
